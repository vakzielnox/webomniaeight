# OmniaEight Control

Plugin WordPress para controlar diseno, movimiento, elementos flotantes, scroll, animaciones, menu, cursor y capas sin modificar el nucleo de WordPress.

## Instalacion en Hostinger

Configura Git deploy hacia esta carpeta:

```text
public_html/wp-content/plugins/omniaeight-control
```

Despues entra en WordPress:

```text
Plugins > OmniaEight Control > Activate
```

Panel de control:

```text
WordPress Admin > OmniaEight
```

## Que controla

- Diseno: color acento, tinte de fondo, profundidad de capas.
- Movimiento: activar/desactivar movimiento, velocidad, reveal on scroll.
- Elementos flotantes: mostrar/ocultar, tamano, opacidad, posicion X/Y.
- Comportamiento: smooth scroll, cursor personalizado, menu sticky.

## Que no debe incluir el repo

```text
wp-admin/
wp-includes/
wp-config.php
uploads/
```

Esas partes pertenecen al nucleo, configuracion privada o archivos subidos por WordPress.
